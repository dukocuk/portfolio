# Project Index: portfolio

Personal portfolio / case-study SPA for **Duran Köse** — bilingual (DA/EN),
dark-only, data-driven. Live: <https://dukocuk.github.io/portfolio/>

> Compact repo map for fast onboarding (generated by `/sc:index-repo`). For depth,
> see [`docs/`](./docs/README.md). Not a build artifact — hand-maintained.

## 📁 Structure

```
index.html              Shell: SEO/OG/JSON-LD meta, fonts, pre-paint lang script
vite.config.ts          base '/portfolio/', react + imagetools plugins
tsconfig*.json          TS project references (root + app)
.github/workflows/      deploy.yml → build + deploy to GitHub Pages on push to main
docs/                   Developer knowledge base (5 topic docs + index)
src/
├── main.tsx            Mounts <App/> in <StrictMode> under <LanguageProvider>
├── App.tsx             Composes all sections in order — no router
├── index.css           Tailwind layers + dark-only CSS-variable design tokens
├── components/         One component per site section (+ ProjectCard, Section)
│   └── ui/             Reusable primitives (Button, Card, Tag, Reveal, Stagger,
│                       ImageGallery, Lightbox, BookingButton, LanguageToggle, …)
├── data/               Bilingual site content — edit here, not the JSX
├── i18n/               config, LanguageContext, useLanguage, ui.ts (chrome strings)
├── hooks/              useScrollSpy, useBodyScrollLock
├── lib/                caseStudyImages, motion, prefetchImage
└── assets/case-studies/{project-id}/   Screenshots — auto-discovered, no registration
```

## 🚀 Entry points

Static browser SPA — no CLI, API, or tests.

- **`index.html`** — HTML shell; inline script sets `<html lang>` from
  `localStorage['lang']` before paint.
- **`src/main.tsx`** — mounts `<App/>` under `<LanguageProvider>`.
- **`src/App.tsx`** — section composition: Navbar → Hero → About → Snapshot →
  Projects → Timeline → Education → Services → Testimonials → Philosophy →
  Contact → Footer (+ BackToTop, FirstVisitModal).

## 📦 Core modules

| Area | Path | Purpose |
|------|------|---------|
| Sections | `src/components/*.tsx` | One component per site section; read copy from `data/` + `i18n/ui.ts` |
| UI primitives | `src/components/ui/` | Button, Card, Tag, Reveal/Stagger, ImageGallery/Lightbox, BookingButton (lazy Cal.com), etc. |
| Content | `src/data/*.ts` | Bilingual `Record<Lang, …>`, `da` default: profile, about, projects, experience, education, services, testimonials, languages, popup |
| Images | `src/lib/caseStudyImages.ts` | Auto-discovers case-study images via `import.meta.glob` + `vite-imagetools` (1600px full / 480px thumb WebP, numeric sort) |
| i18n | `src/i18n/` | `config` (LANGS, DEFAULT_LANG, isLang), `LanguageContext`/`useLanguage`, `ui.ts` chrome strings |
| Hooks | `src/hooks/` | `useScrollSpy` (IntersectionObserver nav) · `useBodyScrollLock` (ref-counted) |
| Motion | `src/lib/motion.ts` | Shared Framer Motion variants (reduced-motion aware) |

## 🔧 Configuration

- **`vite.config.ts`** — `base: '/portfolio/'`; `react()` + `imagetools()` plugins.
- **`tsconfig.json` / `tsconfig.app.json`** — TS project references.
- **ESLint** — flat config (typescript-eslint + react-hooks + react-refresh); no `.eslintrc`.
- **`.github/workflows/deploy.yml`** — builds and deploys to GitHub Pages on push to `main`.
